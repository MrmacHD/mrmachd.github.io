Designs

01 May 2012

After a while of having "no designs available yet." in the designs section of the site, I decided to fix that by displaying some of my recent 2D and 3D CGIs (computer generated imagery).

They were originally downloaded to the user's machine, each time the page was loaded. Then, I changed it to be streamed down to the user's browser, which allows me to use less memory on the server side. Now, I might add some additional interactivity and caching mechanism to reduce page load times and perhaps save some bandwidth.

Hope you enjoy the new designs. I will add more soon, so stay tuned.
