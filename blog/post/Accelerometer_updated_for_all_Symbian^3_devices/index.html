Accelerometer updated for all Symbian^3 devices

20 Apr 2012

This update was long overdue. As I hinted in an earlier post, since I published the iOS version of Accelerometer, I was thinking about bringing some of its features back to the Symbian version of the application.

Now, this update comes only to Symbian^3 devices. Unfortunately, I won't be updating the S60 5th Edition of Accelerometer for the time being. However, I might write more about it specifically at a later date.

You can find more information as well as some tips and tricks about the application on its dedicated site. The ability to record acceleration values, a very useful new feature that was suggested by one of the reviewers has been added too. Among the new features is a new Symbian Belle designed icon. Obviously, the entire UI has been redesigned from the ground up too. I am particularly happy that I could cover such a wide range of devices in a single version of the application. Although version 1.0.0 of Accelerometer had the same characteristics, the UI looked particularly out of date on Symbian Belle.

You can update or buy the Accelerometer for any Symbian^3 device from the Ovi Store. Don't forget to rate and review it!
